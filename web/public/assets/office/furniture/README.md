# Office Furniture Pixel Art

This directory contains the unmodified PNG source pack used by the AgentVille office environment.

- **Source:** `Office-Furniture-Pixel-Art/Office-Furniture-Pixel-Art`
- **Provenance:** Office Furniture Pixel Art pack supplied for the AgentVille furniture refresh.
- **Imported:** 2026-07-17
- **License:** Retained as provided with the source pack; no edits or recompression were applied.

`office-background-furniture.png` is a deterministic composition of these sprites. The reusable build tool is
`tools/build-office-background.py` and reads this directory directly.
